# is-x-browser

JavaScript library to detect if it's an X In-app browser

## Installation

```bash
npm i is-x-browser
```

## Usage

```typescript
import isXBrowser from 'is-x-browser';

if (isXBrowser()) {
	console.log('Inside X browser');
} else {
	console.log('Regular browser');
}
```
